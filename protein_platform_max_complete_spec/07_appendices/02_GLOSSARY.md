# Glossary

Canonical data model:
The internal representation used to unify disparate external sources.

Provenance:
Metadata describing where data came from, how it was parsed, and how it was transformed.

Lineage:
The graph of dependencies between raw inputs, derived objects, datasets, features, and model outputs.

Multimodal:
Using more than one kind of information source, e.g. structure + sequence + pathway.

Noise-robust:
Able to identify signal despite missing, conflicting, weak, or partially incorrect information.

Leakage-safe split:
A dataset split designed to avoid train/test contamination through high similarity or duplication.
