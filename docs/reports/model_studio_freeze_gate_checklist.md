# Model Studio Freeze Gate Checklist

The PPI freeze gate is ready only when all of these are true:

- the benchmark trio plus two governed PPI subsets are launchable
- launchability/governance truth comes from the canonical row-to-subset-to-pool chain
- the guided flow is usable end to end without shell help
- blocked Stage 2 lanes are clearly surfaced and measurably real
- no reviewer has an open P1 finding
- the evidence pack includes browser traces, screenshots, compare/export examples, signoff ledger, deferred-items ledger, and limitations ledger
