# Model Studio Do Not Promise List

Support, operators, and stakeholders must not promise:

- that all governed staged rows are launchable
- that review-pending governed subsets are safe for routine study launches
- that PyRosetta or free-state comparison are already launchable
- that the current sequence-embedding lane is a broader standalone embedding backend
- that non-PPI lanes are part of the shipped beta unless they are explicitly promoted
- that public GA readiness has been reached
