# Model Studio Deferred Items Ledger

## Deferred From Controlled External Beta

These items remain visible or planned, but they are not part of the shipped beta lane.

### Scientific / Representation

- `PyRosetta`
  Reason: prototype artifact exists, but native runtime signoff and matrix proof are incomplete.
- `Free-state comparison`
  Reason: pairing rules exist, but true bound/free structure coverage is not yet sufficient for launch.
- `AlphaFold-derived support`
  Reason: no native, review-cleared materialization path.
- `free-state energetics` claims
  Reason: blocked until real Rosetta outputs and paired-state evidence exist.

### Task Scope

- broad non-PPI activation
  Reason: the shipped beta stays PPI-first.
- public protein-ligand expansion beyond a tightly scoped pilot
  Reason: only allowed after the PPI freeze gate.
- nucleic-acid and other task families
  Reason: out of scope for this beta program.

### Modeling / Packaging

- heterograph, CNN, and U-Net lanes
  Reason: not part of the current trustworthy beta surface.
- unreviewed scientific preprocessing or downloaded semantics
  Reason: beta ship requires native-only scientific truth.

## Deferral Rule

An item leaves this ledger only when:

- implementation is real
- runtime truth is documented
- required reviewers sign off
- user-facing blockers and rationale stay in sync
