# Data Mission

The system must determine not only how to download upstream data, but how to best use, combine, analyze, and store it.

Required outputs for each target source:
1. acquisition method
2. schema surface summary
3. relevant fields for platform goals
4. compatibility with proteins / ligands / PPIs / PNAs
5. join keys
6. strengths and weaknesses
7. planning-index storage recommendation
8. canonical-layer population recommendation
9. lazy materialization recommendation
10. source-specific quality caveats

Priority target sources:
- RCSB / PDBe
- UniProt
- AlphaFold DB
- BindingDB
- InterPro
- DisProt
- BioGRID
- IntAct
- EMDB
- Reactome
- motif / structural motif systems
- evolutionary/MSA pipelines
